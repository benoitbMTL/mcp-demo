Welcome to the MCP demo tool

Here is the simplest way to get started.

  • 1️⃣ Start in the Server tab
    Choose a transport and a protocol version, then launch the MCP server.
  • 2️⃣ Switch to the Client tab
    Connect the MCP client with the same transport and same protocol version.
  • 3️⃣ Send your first request
    Choose an action, review the JSON request, and send it.
  • 4️⃣ Explore the Theme tab
    Change the interface style and choose the theme you prefer.
